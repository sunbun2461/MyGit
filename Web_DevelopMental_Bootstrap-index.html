Start Here

This is where we write about what it means to learn to code, and how I learned to code and made this web site. I am working on becoming a developer, I am going to be one some day now matter what.

HTML is the place to start!

Technically it is HTML5, Hyper Text Markup Language, you will become intimately familiar with this "language" and it will become like second nature, to where it feels like you are cheating. Really until you reach JavaScript it will feel like you are cheating and not really learning, it's quite amazing.


After HTML Comes CSS3

Again, you are going to feel a bit like a cheat. There are some more difficult parts though, Grid and FlexBox come to mind right away. CSS3 stands for cascading style sheets. If HTML is the structure of the page, with the paragraphs themselves, or the headings, the layout, then CSS is the styling, the frills, the stuff that makes things look presentable and aesthetically pleasing. Colors, fonts, specific placements and more come from CSS. The hardest part is that each element interacts with the previous element and the one after it. You can get lost in the weeds rather quick without CSS resets (which we will talk about) Have no fear, there is always a way out of the mistakes that can occur using CSS.


JavaScript!

Unfortunately, once you reach the illustrius JavaScript, you will wish you hadn't 'cheated' so much previously. Just Kidding! However this is the first true Language. An object oriented programming language. This separates the men from the boys. Honestly, anyone who can pass a math class can write code in JavaScript, I do believe it takes quite a bit of time though to truly start to understand JavaScript's intimacies.
